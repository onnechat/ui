# Introduction

**`@onne/ui`** is a set of beautifully-designed, accessible React components built on top of Radix UI and Base UI. Designed for modern React applications with Tailwind CSS v4.

## Getting Started

```bash
npm install @onne/ui
```

```css
/* Import the styles in your globals.css */
@import 'tailwindcss';
@import '@onne/ui/styles';
```

```tsx
import { Button } from '@onne/ui';

function App() {
  return (
    <Button variant="primary">
      Get Started
    </Button>
  );
}
```

## Components

Browse the sidebar to explore all available components. Each component page includes live examples, API documentation, and usage guidelines.

## Design Tokens

`@onne/ui` uses CSS custom properties for theming, making it easy to adapt to any design system:

```css
:root {
  --primary: 346 62% 52%;
  --background: 0 0% 98%;
  --foreground: 240 10% 3.9%;
  --radius: 0.5rem;
}

.dark {
  --primary: 346 62% 52%;
  --background: 240 5.9% 8%;
  --foreground: 0 0% 98%;
}
```

Dark mode is supported out of the box via the `.dark` class on any parent element.
